# T-cnicas-de-an-lisis-y-modelado-
Portafolio digital del trabajo en clase 
EL curso de la materia de técnicas de análisis y modelado, busca que el alumno disstinga las técnicas de análisis para el tratamiento y análisis de datos multivariados y no lineales que se emplean para el estudio de las Ciencias Ambientales, a tráves de los siguientes temas, con su respectivo objetivo específico: 
Modelo de regresión lineal
Identificar los fundamentos del modelo de regresión lineal simple y múltiple para explicar relaciones entre variables ambientales continuas.
Regresión de Poisson
Reconocer el modelo de regresión de Poisson como herramienta estadística para el análisis de datos de conteo en fenómenos ambientales, como abundancia de especies o eventos de contaminación.
Regresión logística
Comprender el modelo de regresión logística binaria para clasificar y predecir variables de respuesta dicotómica en contextos ambientales, como presencia/ausencia de una especie.
Regresión multinomial
Aplicar el modelo de regresión logística multinomial para el análisis de variables de respuesta con múltiples categorías en estudios ambientales, como tipos de cobertura vegetal o uso de suelo.
Ecuaciones diferenciales
Comprender el uso de ecuaciones diferenciales ordinarias y en derivadas parciales para modelar dinámicas ambientales, como crecimiento poblacional, dispersión de contaminantes y flujos de energía.
Modelos de redes
Analizar la estructura y aplicación de los modelos de redes para representar interacciones ecológicas, flujos de materia y conectividad en sistemas ambientales complejos.
